## Features
- Checks for minimum length (8 characters).
- Detects Uppercase and Lowercase letters.
- Validates the presence of numbers and special characters.
- Provides a real-time score and feedback.

## How to Run
1. Ensure you have Python installed.
2. Clone this repo: `git clone https://github.com/Shamyuktha-07/PRODIGY_CS_03.git`
3. Run the script: `python passwordcheck.py`

## Logic Flow
The tool uses Python's `re` module to perform pattern matching and calculates a score out of 5.
